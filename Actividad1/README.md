PROYECTO BIBLIOTECA

Sistema de gestión para una biblioteca, construido sobre una base de datos NoSQL orientada a documentos (MongoDB).

PROBLEMA ESCOGIDO

Una biblioteca  necesita controlar su catálogo de libros y autores, además de llevar el seguimineto de los préstamos que realizan sus usuarios. Esta información se dispersa en registros manuales o en hojas de cálculo, lo que dificulta saber en tiempo real la disponibilidad de un libro.
